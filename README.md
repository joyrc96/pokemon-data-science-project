# pokemon-data-science-project
An exploratory data science project where I mess around with pre-processing and various ML algorithms using the Pokemon dataset by Sanjana Chaudhari on Kaggle. 

## Description

This is a simple project where I experiment with different EDA and ML methods, plotting graphs, and drawing conclusions from data. I'm aiming to create more projects like these to get more used to both formatting code to an industry standard, and truly understanding these algorithms at a deeper level, knowing the pros and cons of each, what each algorithm is suited for, and knowing what the results of each one actually means. I'm also learning how to format README.md files, document my code properly, and generally format projects in a professional way.
